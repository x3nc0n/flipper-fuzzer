### Build Fixes

1. **platformio.ini** — Removed `-DARDUINO_USB_CDC_ON_BOOT=1`
   - Root cause: espressif32 3.20017.241212 framework bug; HWCDC replaces Serial but HardwareSerial.cpp doesn't reference it
   - Trade-off: debug output now shared with Flipper UART link (UART0)

2. **main.cpp** — Replaced all `Serial0` with `Serial`
   - Serial0 undefined without USB CDC boot flag
   - All prints (debug + protocol) now use UART0

### Flash Procedure

Manual bootloader entry required (no DTR/RTS auto-reset over ESP32-S2 native USB):
1. Hold BOOT button
2. Tap RESET (while holding BOOT)
3. Release BOOT
4. Run: `python -m platformio run -t upload --upload-port COM8`

ROM bootloader enumerated as VID_303A/PID_0002 on COM8; app mode uses COM6/COM7.

All 4 partitions verified on chip.
